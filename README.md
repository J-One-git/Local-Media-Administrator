# Local Media Administrator (LMA)

Rediscover your own video archive.

LMA is a lightweight Windows app for browsing large local video libraries with a random feed and shortform mode.

## Features
- Random feed for large libraries
- Shortform mode for shorter clips
- Tags & playlists
- Fully offline (your files never leave your computer)
